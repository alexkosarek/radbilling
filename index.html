How it works

  1. Upload your monthly case export from your RIS system (.xlsx or .csv).
  2. Run billing — every CT read is valued at $35, every MRI at $45. Duplicates (accessions already billed in a prior month) are automatically removed.
  3. Download the report — a clean Excel file with all case info plus a Compensation column and total at the bottom. All new accessions are saved for future duplicate checking.
  4. Tax season: use the "Download all-time report" button to get every case ever billed as a single Excel file.
